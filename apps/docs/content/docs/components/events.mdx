## Features

### Event Features

- **Event Title**: Display the main title of the event.
- **Event Subtitle**: Provide additional information about the event.
- **Event Icon**: Add an icon to visually represent the event.
- **Event Date**: Show the date of the event.
- **Event Time**: Indicate the time when the event will take place.
- **Event Status**: Display the current status of the event (e.g., upcoming, ongoing, completed).

These features allow you to create a comprehensive and visually appealing event list, making it easier for users to understand and keep track of various events.
