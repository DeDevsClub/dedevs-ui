## Features

- **Multi-Tab Interface**: Convert, Buy, and Send operations in a single component
- **Token Selection**: Dropdown with coin icons and names
- **Real-time Conversion**: Live amount calculations with formatted numbers
- **Balance Display**: Shows available token balances
- **Transaction Summary**: Detailed breakdown of fees and totals
- **Modern Design**: Clean, professional UI with dark mode support
- **Accessibility**: Full keyboard navigation and screen reader support

## Component Structure

The Converter component includes:

- **Tab Navigation**: Convert, Buy, Send modes
- **Converter Fields**: Input/output token selection and amounts
- **Transaction Summary**: Fees, totals, and exchange rates
- **Settings Button**: Additional configuration options

## Token Configuration

The component uses a coins array for token selection:

```tsx
const coins = [
  {
    id: "1",
    name: "Ark",
    image: "https://example.com/ark-icon.svg"
  },
  {
    id: "2", 
    name: "Tok",
    image: "https://example.com/tok-icon.svg"
  }
];
```
