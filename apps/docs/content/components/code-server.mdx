## Features

- **Server-Side Rendering**: Code highlighting on the server for better performance
- **Shiki Integration**: High-quality syntax highlighting with VS Code themes
- **Zero Client JS**: No JavaScript required on the client side
- **Theme Support**: Multiple built-in themes and custom theme support
- **Language Support**: 100+ programming languages supported
- **Performance**: Faster initial load with pre-rendered syntax highlighting

## Usage

The Code Server component renders syntax-highlighted code blocks on the server, eliminating the need for client-side highlighting libraries and improving performance.

### Basic Usage

```tsx
import { CodeBlockContent } from '@repo/code/server';

export async function CodeExample() {
  const code = `
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
`;

  return (
    <CodeBlockContent
      language="javascript"
      themes={{ dark: 'github-dark', light: 'github-light' }}
    >
      {code}
    </CodeBlockContent>
  );
}
```

### With Custom Theme

```tsx
import { CodeBlockContent } from '@repo/code/server';

export async function ThemedCodeBlock() {
  const pythonCode = `
def quicksort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quicksort(left) + middle + quicksort(right)
`;

  return (
    <CodeBlockContent
      language="python"
      themes={{ dark: 'dracula', light: 'github-light' }}
      className="rounded-lg"
    >
      {pythonCode}
    </CodeBlockContent>
  );
}
```

### Multiple Code Blocks

```tsx
import { CodeBlockContent } from '@repo/code/server';

export async function MultipleCodeBlocks() {
  const examples = [
    { code: 'console.log("Hello World");', language: 'javascript' },
    { code: 'print("Hello World")', language: 'python' },
    { code: 'puts "Hello World"', language: 'ruby' }
  ];

  return (
    <div className="space-y-4">
      {examples.map((example, index) => (
        <CodeBlockContent
          key={index}
          language={example.language}
          themes={{ dark: 'github-dark', light: 'github-light' }}
        >
          {example.code}
        </CodeBlockContent>
      ))}
    </div>
  );
}
```

## Props

```tsx
interface CodeBlockContentProps {
  children: string;
  language?: BundledLanguage;
  themes?: CodeOptionsMultipleThemes['themes'];
  className?: string;
  syntaxHighlighting?: boolean;
}
```

### Prop Details

- **children**: The source code to highlight (required)
- **language**: Programming language for syntax highlighting
- **themes**: Shiki theme configuration object for dark/light modes
- **className**: Additional CSS classes
- **syntaxHighlighting**: Enable/disable syntax highlighting (default: true)

## Advanced Features

### Theme Configuration

```tsx
<CodeBlockContent
  language="typescript"
  themes={{
    dark: 'github-dark',
    light: 'github-light'
  }}
>
  {complexCode}
</CodeBlockContent>
```

### Disable Syntax Highlighting

```tsx
<CodeBlockContent
  language="python"
  syntaxHighlighting={false}
>
  {plainTextCode}
</CodeBlockContent>
```

### Custom Styling

```tsx
<CodeBlockContent
  language="javascript"
  className="border-2 border-blue-500 rounded-xl p-4"
  themes={{ dark: 'github-dark', light: 'github-light' }}
>
  {code}
</CodeBlockContent>
```

## Best Practices

1. **Choose Appropriate Themes**: Match your site's design system
2. **Use Line Numbers**: For longer code blocks and tutorials
3. **Highlight Important Lines**: Draw attention to key concepts
4. **Optimize for Mobile**: Ensure code blocks are readable on small screens
5. **Cache Rendered Output**: Use static generation for better performance
6. **Consistent Styling**: Use the same theme across your site
