- **Animated Entrance**: Smooth fade-in animations using Framer Motion
- **Flexible Content**: Support for title, subtitle, description, and features

- **Background Support**: Custom background images with overlay

- **Accessibility**: Full keyboard navigation and screen reader support

## Component Structure

The Hero component includes:

- **Main Content**: Title, subtitle, and description with animations
- **Action Buttons**: Primary and secondary CTAs with hover effects
- **Features List**: Animated feature highlights
- **Testimonial Section**: Customer quote with author details
- **Background Layer**: Optional background image with overlay

## Usage Examples

### Basic Hero

```tsx
import { Hero } from '@repo/site/hero';

export function LandingHero() {
  return (
    <Hero
      title="Build Amazing Applications"
      description="Create stunning user interfaces with our comprehensive component library designed for modern web applications."
      primaryAction={{
        label: "Get Started",
        href: "/docs"
      }}
      secondaryAction={{
        label: "View Components",
        href: "/components"
      }}
    />
  );
}
```

### With Features

import { Hero } from '@repo/site/hero';

## Props Interface

```tsx
interface HeroProps {
  title: string;
  subtitle?: string;
  description: string;
  primaryAction?: {
    label: string;
    href?: string;
    onClick?: () => void;
  };
  secondaryAction?: {
    label: string;
    href?: string;
    onClick?: () => void;
  };
  features?: string[];
  testimonial?: {
    text: string;
    author: string;
    role: string;
    avatar?: string;
  };
  backgroundImage?: string;
  className?: string;
}
```

## Animation Features

The component uses Framer Motion for smooth animations:

- **Staggered Entrance**: Elements animate in sequence
- **Fade In Up**: Smooth upward motion with opacity transition
- **Hover Effects**: Interactive button and feature animations
- **Responsive Timing**: Optimized animation timing for different screen sizes

## Styling

- **Typography**: Responsive text sizing with proper hierarchy
- **Spacing**: Consistent spacing using Tailwind utilities
- **Colors**: Semantic color usage with dark mode support
- **Shadows**: Subtle shadows and gradients for depth
- **Responsive**: Mobile-first responsive design

## Accessibility

- Proper heading hierarchy (h1 for title)
- Screen reader friendly content structure
- Keyboard navigation support
- High contrast color schemes
- Focus management for interactive elements

## Integration

Perfect for:

- **Landing Pages**: Main hero section for product pages
- **Marketing Sites**: Feature announcements and promotions
- **Documentation**: Getting started sections
- **Portfolio Sites**: Personal or company introductions
- **SaaS Applications**: Product showcases and onboarding

## Customization

The component can be customized by:

- Modifying animation timing and effects
- Customizing color schemes and typography
- Adding additional content sections
- Integrating with routing libraries
- Adding analytics tracking to CTAs
